# Journey

Extension to manage and share your web journey.

---

## Install from Source

- Clone/Download this repo.
- Extract the zip file.
- Go to `chrome://extensions/`.
- Enable `Developer mode` switch.
- Click on `Load unpacked`.
- Select the extracted-folder.
- Pin it if you want.
- Done.
